# Proposals

This folder contains proposals submitted by the team via **"FEEDBACK DE USO"** and **"DESCUBRIMIENTO"** flows in Claude Code.

Each file is a single `.md` document with:
- The problem or discovery described by the developer
- A proposed solution or context update (as a prompt ready to apply)

## Naming convention

```
{github-username}-{short-slug}.md
```

## How proposals get here

1. A developer says "FEEDBACK DE USO" or "DESCUBRIMIENTO" in their Claude Code session
2. The AI guides them through the template and builds the `.md` file
3. A PR is created with that single file
4. The team reviews and votes
5. When merged, the proposal lands here as a permanent record
